# Task 1 — Build Your First Real Skill

## Skill Name

**Rough Idea Planner**

## Task Overview

In this task, I created a custom AI skill that helps me turn rough project ideas into clear and structured plans. The purpose of this skill is to make my daily project planning easier and more consistent.

## Why I Chose This Skill

I often get rough ideas for websites, apps, assignments, business tasks, and software projects. Sometimes those ideas are not properly organized. This skill helps me convert those rough thoughts into a proper plan with clear steps, features, timeline, risks, and next actions.

## What the Skill Does

The skill takes a simple idea and formats it into a structured project plan. It can be used for software project ideas, university assignments, business ideas, feature planning, and personal productivity plans.

## Trigger Phrases

- “I have an idea...”
- “Turn this idea into a plan”
- “Make a project plan for this”
- “Structure this idea”
- “Help me plan this project”

## Output Format

1. Project Title
2. One-Line Summary
3. Problem
4. Target Users
5. Main Goal
6. MVP Features
7. Step-by-Step Plan
8. Suggested Tools or Technology
9. Timeline
10. Risks or Challenges
11. Next 3 Actions

## Test Prompt

```text
I have an idea to build a small website where students can upload assignments and track their progress. Turn this idea into a plan.
```

## Expected Result

The skill should automatically understand the request and return a clear structured plan without needing a special command.

## Learning Outcome

From this task, I learned how to create a reusable AI skill that follows my preferred format and gives consistent results every time.
