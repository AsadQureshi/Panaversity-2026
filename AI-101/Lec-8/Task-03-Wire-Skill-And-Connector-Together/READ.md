# Task 3 — Wire Skill + Connector Together

## Skill Used

**Rough Idea Planner**

## Connector Used

**Indeed Connector**

## Task Overview

In this task, I combined my custom skill with a connected app. I used the Rough Idea Planner skill together with the Indeed connector to fetch live job data and format it into a structured application plan.

## Purpose of the Task

The purpose of this task was to create a workflow where one normal sentence could fetch real data from a connected app and automatically format it using my skill.

## Prompt Used

```text
Use Indeed to find Senior Laravel Developer or Senior PHP Laravel jobs posted in the last 2 days, then turn the best matches into a structured application plan with priority order, reasons, required skills, risks, and next 3 actions.
```

## How It Worked

The Indeed connector retrieved real job listings. Then my Rough Idea Planner skill organized the information into a clear plan for which jobs I should apply to first.

## Expected Output

The final result included priority order of jobs, reasons for applying, important skills required, possible risks or challenges, suggested next actions, and a structured plan for applying.

## Spot Check

I checked one job listing from the result and compared the job title, company name, location, posted date, and apply link with the original Indeed source.

## Learning Outcome

From this task, I learned how a skill and connector can work together in one workflow. I also learned that AI can fetch live data and organize it into my preferred format without manual copy-pasting.
